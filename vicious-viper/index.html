// features

What's Included

More than just config files — a complete, opinionated workflow for daily Arch Linux use.

Keyboard-driven layout — Navigate workspaces and windows entirely without a mouse using ALT as the modifier

Dynamic theming via Matugen — Wallpaper-driven Material You palettes applied to every app automatically

Smart installer with flags — --dry-run, --select, --uninstall and timestamped backups

Laptop-optimized — auto-cpufreq, battery module, brightness and touchpad gestures tuned out of the box

yt-dlp + aria2 integration — Download videos, audio, and manage downloads via custom scripts

AirPods + Bluetooth TUI — Dedicated menus for Bluetooth devices and AirPods management

Waybar config — Clean, minimal status bar with workspaces, clock, battery, and system tray

GTK + Qt themes — .themes directory included for consistent GUI app theming

Clipboard + emoji pickers — cliphist history and rofiemoji bound to quick shortcuts

Works on desktops too — Primarily laptop-focused but fully desktop-compatible with minor config tweaks

// theming

Dynamic Color System

Powered by Matugen — a Material You color extraction engine that derives a full palette from your active wallpaper and recolors every component automatically.

Material You
How It Works

Set a wallpaper with aww, then run matugen image on it. Matugen reads the dominant colors, generates a full Material You token set, and writes them into Waybar, Rofi, Mako, Hyprlock, GTK apps, and the terminal — all in one command.

$ aww set ~/walls/forest.jpg
$ matugen image ~/walls/forest.jpg
Themed Components
Waybar status bar
Rofi app launcher
Mako notifications
Hyprlock lock screen
GTK apps + terminal

Templates live in .config/matugen/ — edit them to customize per-app color mapping.

// install

Installation

Run --dry-run first to preview every action before committing. Your existing configs are backed up automatically.

Recommended: Always run --dry-run first. The installer backs up every file it overwrites to ~/.local/share/hypr-dotfiles-backups/<timestamp>/. Use --uninstall to restore originals precisely.

Clone the repository

$ git clone https://github.com/Cybersnake223/Hypr
$ cd Hypr && chmod +x install.sh

Preview changes (recommended)

$ ./install.sh --dry-run

Run the installer

# Interactive module picker:
$ ./install.sh --select
# Or install everything non-interactively:
$ ./install.sh --yes

Undo if needed

$ ./install.sh --uninstall

Copy config files

$ cp -r .config/* $HOME/.config
$ cp -r .local/bin/scripts $HOME/.local/bin
$ cp -r .icons $HOME/.icons
$ cp -r .themes $HOME/.themes
$ cp -r .fonts $HOME/.fonts
$ cp .Xresources .gtkrc-2.0 $HOME/

Mark scripts executable & rebuild font cache

$ find $HOME/.local/bin/scripts -type f -exec chmod +x {} +
$ fc-cache -f

Add scripts to PATH

# zsh / bash:
$ echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.zshrc
# fish:
$ fish_add_path $HOME/.local/bin
--dry-run
Preview every action — zero changes made. Run this first.
--select
Interactive TUI to toggle which modules to install.
--yes
Non-interactive, auto-confirm all prompts.
--uninstall
Restore originals from the most recent backup precisely.
--list-backups
Show all backups with timestamps, file counts, and sizes.
--skip-deps
Skip the ecosystem dependency check (faster re-installs).
--no-backup
Skip backup step. Dangerous — disables --uninstall.
-h / --help
Show usage and all available options.
